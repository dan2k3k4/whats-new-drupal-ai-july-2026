- 🏕 **[README-DEMO.md](README-DEMO.md)** — what's on the site, rebuild-from-scratch steps, asset credits
- 🎬 **[DEMO-RUNBOOK.md](DEMO-RUNBOOK.md)** — stage runbook: tab URLs, per-demo do-tables, panic sheet

and `ai_best_practices` — all running on the amazee.ai provider.
